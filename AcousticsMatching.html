Acoustics Matching

This page gives some demo audio examples for Acoustics Matching in reverberant environments. Each file consists of a concatenation of two audio signals. The first part is the reference signal. The second part is either the original input or the processed input.

"Before": Concatenation of Reference and Original Input

"After": Concatenation of Reference and Matched Input

The listener is expected to answer the following questions. To do so, please provide a score in the range of (0 - 10) for each of the following questions.

  • Question 1:Rate the quality of the match between "Before" and "After" cases.
  • Question 2: Rate the quality of the "After" sound for distortions and abnormalities.
